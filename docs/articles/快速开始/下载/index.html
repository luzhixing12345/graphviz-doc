下载

https://graphviz.org/download/

graphviz的下载比较方便

  • Ubuntu/Debian
    sudo apt install graphviz
  • Windows

    windows提供了稳定的安装包,您需要前往graphviz-windows下载最新版本的graphviz的exe安装程序

    将bin/加入环境变量即可全局调用

    如果有choco的话

    choco install graphviz
  • MacOS
    brew install graphviz

下载完成后检查dot版本,成功输出版本信息及帮助即成功

dot -V
